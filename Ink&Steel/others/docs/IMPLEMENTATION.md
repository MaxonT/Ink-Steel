## 使用说明

1. **添加钢笔数据**：编辑 `data/pens.json` 文件
2. **导入数据**：使用 `utils/data-import.js` 中的工具
3. **查看钢笔**：访问首页浏览所有钢笔
4. **搜索过滤**：使用首页的搜索和过滤功能
5. **查看详情**：点击钢笔卡片查看详细信息
6. **对比钢笔**：在详情页或首页点击"Add to Comparison"
7. **查看对比**：访问 `/compare.html` 查看对比结果
