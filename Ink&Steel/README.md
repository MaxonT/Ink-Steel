# Ink & Steel - 钢笔图鉴网站

一个美观优雅的钢笔大合集网站，包含详细的钢笔信息（型号、类型、品牌、尺寸、图片等）和购买链接。

## 项目特点

- 📝 超级详细的钢笔信息（尺寸、规格、材料等）
- 🖼️ 多角度图片展示
- 🔗 购买链接管理
- 🔍 高级搜索和过滤功能
- 🎨 优雅美观的用户界面
- 📊 品牌浏览和统计

## 数据结构

详细的数据结构请参考 `data/pens.json` 和 `data/inks.json` 文件。

## 开发

这是一个静态网站项目，使用：
- HTML5
- Vanilla JavaScript (ES6+)
- Web Components
- Tailwind CSS
- JSON 数据存储

## 数据来源

数据可以从以下来源获取：
- Anna's Archive
- 品牌官方网站
- 在线零售商
- 钢笔数据库
